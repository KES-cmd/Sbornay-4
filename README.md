# Поиск изображений по текстовому описанию

## Описание проекта
Система для определения соответствия между текстовыми описаниями и изображениями. Модель принимает на вход векторное представление изображения и текста, а на выходе выдает вероятность их соответствия (score от 0 до 1).

## Основные этапы

### 1. Подготовка данных
- Загрузка и объединение экспертных и краудсорсинговых оценок
- Создание финальной целевой переменной (взвешенное среднее 70/30)
- Фильтрация запрещенного контента (изображения с детьми)
- Очистка от дубликатов и пропусков

### 2. Векторизация
**Тексты:** TF-IDF с 2000 признаков
- Максимальная размерность: 2000
- N-gram диапазон: (1, 2)
- Английские стоп-слова удалены

**Изображения:** ResNet-18
- Предобученная на ImageNet
- Удален последний классификационный слой

### 3. Модель
**Алгоритм:** Gradient Boosting Regressor
- Целевая переменная: вероятность соответствия (0-1)
- Метрика: MSE/RMSE
- Размерность входных данных: 3000 признаков (2000 текст + 1000 изображение)

### 4. Тестирование
- Поиск топ-4 наиболее релевантных изображений для текстового запроса
- Проверка на запрещенный контент с выводом дисклеймера
- Визуализация результатов поиска

## Результаты
- Успешно реализован полный пайплайн от данных до предсказаний
- Добавлена фильтрация запрещенного контента
- Протестировано 3 типа моделей, лучшей оказалась Gradient Boosting
